# BudgetBuddy

A personal budget tracker Android app built for the OPSC6311 Part 2 POE.

## Features
- Login/Register with local database
- Create and manage expense categories (with colours & icons)
- Add expense entries with date, time, description, and optional receipt photo
- Set monthly minimum and maximum spending goals
- View expenses filtered by custom date range
- View category totals for any period
- Dashboard with overall budget progress, category breakdown, and pie chart
- Export expense list to PDF
- Built‑in currency converter (offline)
- Gamification placeholder (badges)
